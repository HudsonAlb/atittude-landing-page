# Atittude Gym - Landing Page

![Atittude Gym Logo](https://images.unsplash.com/photo-1534438327276-14e5300c3a48?q=80&w=1200&auto=format&fit=crop)

Uma Landing Page moderna, responsiva e de alta performance desenvolvida para a academia **Atittude**. Este projeto foca em conversão e experiência do usuário, utilizando tecnologias de ponta para garantir fluidez e estética premium.

## 🚀 Tecnologias Utilizadas

- **React 18**: Biblioteca principal para construção da interface.
- **Vite**: Build tool ultra-rápida para um desenvolvimento ágil.
- **Framer Motion**: Responsável pelas animações fluidas e micro-interações.
- **CSS3 (Vanilla)**: Design system customizado utilizando variáveis para fácil manutenção.
- **Font Awesome**: Ícones modernos instalados via CDN.
- **Google Fonts**: Tipografia impactante (Montserrat e Bebas Neue).

## ✨ Funcionalidades

- **Dark Mode Premium**: Estética sofisticada em preto e amarelo neon.
- **Design Responsivo**: Adaptado para Desktop, Tablet e dispositivos Mobile.
- **Mural de Avisos Animado**: Sistema rotativo de notícias com transições suaves.
- **Seção de Diferenciais**: Grid interativo com hover effects e imagens de alta qualidade.
- **Tabela de Planos**: Comparativo de tipos de matrícula com destaque para o plano Pro.
- **Galeria da Comunidade**: Mural de fotos com filtros dinâmicos.
- **Formulário de Contato**: Integrado e pronto para captar leads.

## 📦 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone git@github.com:HudsonAlb/atittude-landing-page.git
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd atittude-landing-page
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Para build de produção:**
   ```bash
   npm run build
   ```
